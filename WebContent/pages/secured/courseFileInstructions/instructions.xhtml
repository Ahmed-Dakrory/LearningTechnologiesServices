Course Files confidentiality

  • A course file typically includes sensitive, confidential student information.
  • The TAs shouldn’t access the course file as it has sensitive data such as the students’ final exam samples and the students’ final scores and grade letters.
  • For the Final Assessments (Final Exams) samples, please cover the student name and ID before adding them.

Notes

  • The TAs can help scan/upload the samples ( except the final assessment/ exam  that should be done by the instructor).
  • Please use the latest version of CASh
  • If you are using an older version of CASh , import the old file data to the new one, using the Import Data button.
