Drop Policy:

Drop / Add Week (1st Week)
Deadline: 19 February, 2015
Refundable: Yes
Approvals: Dean, Program Advisor, Registrar
Grade: No grade, and will not be added on transcript

Drop / Withdrawal without Academic Penalty (First 6 Weeks)
Deadline: 28 March, 2015
Refundable: No
Approvals: Dean, Program Advisor, Instructor, Registrar
Grade: "W" (On Transcript)

Drop / Withdrawal with Academic Penalty (1 Month before the end of Semester)
Deadline: 7 May, 2015
Refundable: No
Approvals: Dean, Program Advisor, instructor Registrar
Grade: "WF" or "WP" – This will be based on professor evaluation of term performance (On Transcript)

In the last month of the semester no drops or withdrawals will be approved. If the course requirements are not satisfied the student will receive an "F" grade to be added to the transcript.​

* Drop = Drop from one course or more    

*Withdrawal = Dropping all courses and leaving the university.
